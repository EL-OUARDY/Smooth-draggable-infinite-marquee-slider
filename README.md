# Smooth Draggable Infinite Marquee Slider

A smooth, infinite, draggable marquee slider implementation using GSAP (GreenSock Animation Platform).

## Description

This project demonstrates a horizontal slider that loops infinitely. Users can drag the slider to navigate through the items with smooth inertia physics. It is built with vanilla JavaScript and leverages the power of GSAP for animations and interactions.

## Features

- **Infinite Loop**: The slider content loops seamlessly.
- **Draggable**: Users can click and drag to move the slider.
- **Smooth Physics**: Includes inertia for a natural throwing feel.
- **Responsive**: Adapts to the container width.
- **Active State**: Highlights the center/active element.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- [GSAP](https://greensock.com/gsap/) (Core, Draggable, InertiaPlugin)

## Setup & Usage

1.  Clone or download the repository.
2.  Open `index.html` in your web browser.
3.  Drag the slider to interact with it.
